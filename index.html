IVTV Player

Player for PVR TV cards

Download .zip Download .tar.gz View on GitHub

Description

This is suite of programs for PVR TV cards based on IVTV driver. It containts TV player, radio player and GUI fo V4L2 controls. The players have support for LIRC and XOSD.

Requirements

  • PVR TV card
  • IVTV driver and utilities (v4l2-ctl, ivtv-radio, ivtv-tune) [http://www.ivtvdriver.org]
  • MPlayer (mplayer) [http://www.mplayerhq.hu]
  • ALSA utils (aplay) [http://www.alsa-project.org/]
  • Perl (Lirc::Client, XML::Simple, Gtk2; optionaly: X::Osd)
  • XML Syntax Validator (xmllint, jing, ...) = optional
